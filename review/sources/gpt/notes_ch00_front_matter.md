# Chapter 0: Front Matter - Review Notes

## Overall Assessment
The front matter is structurally sound and conventional. It provides useful context for the thesis through the title page, dedication, epigraph, abstract, acknowledgements, contents, and list of figures.

There are only a small number of issues here, but a few are clear correctness problems worth fixing.

---

## Issues

1. **p6**: "Firstly, my supervisor Martin Hendry for providing me with the opportunity..." is a sentence fragment. It needs a verb, for example: "Firstly, I thank my supervisor Martin Hendry for providing me with the opportunity..."

2. **p6**: "and hence conserve key physical values" has a grammar/agreement problem. Likely intended alternatives are "and hence conserved key physical values" or "thereby conserving key physical values."

3. **p8**: The contents includes `4.0.1 Linear Growth Factor`. That numbering is unusual and looks like a likely sectioning or ToC bug unless it is very deliberately intended.

4. **PDF metadata**: The PDF has blank `Title`, `Author`, and `Subject` metadata fields. This does not affect the visible thesis, but it is a polish and archiving issue.

5. **p1 vs p2**: The repository wrapper page uses "Schrodinger" while the thesis title page uses "Schrödinger". If the wrapper page is under your control, it would be better to make these consistent.

---

## Minor Style / Polish

6. **p6**: "fourier transforms" should be "Fourier transforms".

7. **p6**: "thought provoking" would usually be hyphenated as "thought-provoking".
